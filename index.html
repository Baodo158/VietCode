VietCode

VietCode is a Vietnamese-themed programming language designed for beginners. It uses simple, readable Vietnamese syntax to help new coders understand basic programming concepts easily.

Supports

  • Variable assignment: nhận (assign)
  • Output: in (print)
  • Conditionals: nếu (if), vậy (else), kết thúc nếu (end if)
  • Loops: lặp từ (loop from), đến (to), trong khi (while)

Example Code

FizzBuzz Program:

    i nhận 1
    lặp từ i đến 15:
        chia3 nhận i % 3
        chia5 nhận i % 5
        nếu chia3 bằng 0:
            nếu chia5 bằng 0:
                in "FizzBuzz"
            kết thúc nếu
            nếu chia5 khác 0:
                in "Fizz"
            kết thúc nếu
        kết thúc nếu
        nếu chia3 khác 0:
            nếu chia5 bằng 0:
                in "Buzz"
            kết thúc nếu
            nếu chia5 khác 0:
                in i
            kết thúc nếu
        kết thúc nếu
    kết thúc lặp
