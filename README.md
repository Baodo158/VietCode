## Overview of Keywords

- nhận  | assign
- in  | print 
- nếu  | if
- vậy  | else
- kết thúc nếu  | end if 
- lặp từ  | loop from
- đến  | to
- kết thúc lặp  | end loop 
- trong khi  | while
- kết thúc trong khi  | end while 
- cộng  | add | +
- trừ  | subtract | -
- nhân  | multiply | *
- chia  | divide | /
- %  | modulus | %
- lớn hơn  | greater than | >
- bé hơn  | less than | <
- bằng  | equal to | =
- khác  | not equal to | !=
- //  | comment
- " "  | string
